# TickFlow ⚡
A lightweight, Folia-compatible optimization plugin designed to squeeze maximum performance out of any hardware. TickFlow targets the biggest causes of server lag without ruining the vanilla survival experience.

### 🚀 Features:
* **Smart Entity Limits:** Restricts the maximum number of monsters and dropped items per chunk to prevent entity lag and server crashes.
* **Projectile Cleanup:** Instantly removes useless skeleton arrows upon hitting a block, while safely preserving player-shot arrows and tridents.
* **Auto-Lobotomy for Villagers:** Automatically detects villagers trapped in trading halls or iron farms (e.g., inside vehicles or 1x1 boxes) and disables their pathfinding AI to heavily reduce CPU load. Free-roaming villagers remain completely unaffected.

### ⚙️ Compatibility
* Native support for **Folia** and **Paper**.
